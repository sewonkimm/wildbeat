# Wild Beat Music Academy - 방문자 여정 맵 (User Journey Map)

## 1. 페르소나 정의

### 페르소나 A: 입시 준비생 (김하늘, 18세)

- **배경**: 고등학교 2학년, 실용 음악과 진학 희망
- **목표**: 입시에 특화된 레슨과 합격 실적이 검증된 학원 찾기
- **고민**: "합격률이 높은 학원인지", "강사 경력이 충분한지", "커리큘럼이 체계적인지"
- **의사결정 기준**: 입시 합격 실적, 강사 전문성, 커리큘럼 구체성
- **유입 채널**: 네이버 검색 ("실용음악 입시 학원"), 입시 커뮤니티, 지인 추천

### 페르소나 B: 취미 수강생 (박서준, 27세)

- **배경**: 직장인, 퇴근 후 기타 배우고 싶음
- **목표**: 부담 없이 즐길 수 있는 취미 레슨 찾기
- **고민**: "초보자도 가능한지", "시간이 유연한지", "분위기가 편한지"
- **의사결정 기준**: 접근성(위치/시간), 분위기, 수강료, 수강 후기
- **유입 채널**: 네이버/구글 검색 ("기타 레슨 [지역명]"), 인스타그램, 네이버 플레이스

### 페르소나 C: 학부모 (이정희, 45세)

- **배경**: 중학생 자녀가 보컬에 관심, 향후 입시도 고려
- **목표**: 믿을 수 있는 학원에 자녀를 보내기
- **고민**: "교육 환경이 안전한지", "체계적인 커리큘럼인지", "비용 대비 가치가 있는지"
- **의사결정 기준**: 학원 신뢰도, 시설, 강사 자격, 학부모 후기
- **유입 채널**: 맘카페, 네이버 검색, 지인 추천, 블로그

## 2. 유입 경로별 랜딩 시나리오

### 검색 유입 (Search)

| 검색 키워드 | 랜딩 페이지 | 기대 행동 |
|------------|------------|----------|
| "실용음악 입시 학원" | `/` 또는 `/courses` | 입시반 커리큘럼 확인 → 합격 실적 확인 → 상담 신청 |
| "보컬 레슨 [지역명]" | `/courses` | 보컬 커리큘럼 확인 → 강사 확인 → 상담 신청 |
| "기타 학원 추천" | `/` | 학원 하이라이트 확인 → 수강 후기 → 상담 신청 |
| "와일드비트 음악학원" | `/` | 브랜드 검색 — 전체 탐색 후 상담 |
| "[학원명] 수강료" | `/courses` 또는 `/contact` | 수강료 안내 확인 → 상담 신청 |

### SNS 유입 (Social)

| 채널 | 유입 콘텐츠 | 랜딩 페이지 |
|------|-----------|------------|
| 인스타그램 | 수강생 공연 영상, 레슨 현장 | `/` 또는 `/instructors` |
| 유튜브 | 강사 커버 영상, 레슨 하이라이트 | `/instructors` |

### 추천 유입 (Referral)

| 경로 | 기대 행동 |
|------|----------|
| 지인 추천 (URL 공유) | `/` → 전체 탐색 → 상담 신청 |
| 맘카페/커뮤니티 링크 | `/about` 또는 `/courses` → 신뢰도 확인 → 상담 신청 |

## 3. 전환 퍼널 (Conversion Funnel)

### 퍼널 단계

```
[인지]         [탐색]          [비교/평가]       [의사결정]       [전환]
Awareness → Exploration → Evaluation → Decision → Conversion

검색/SNS     사이트 탐색     커리큘럼/강사     "여기 괜찮다"    상담 신청 폼
유입         콘텐츠 소비     후기 확인        판단 완료        제출
```

### 페르소나별 퍼널 여정

#### 페르소나 A: 입시 준비생

```
[인지] 네이버 "실용음악 입시 학원" 검색
  ↓
[탐색] / (랜딩) → 입시 합격률 수치 확인 → 수강생 성과 확인
  ↓
[비교] /courses → 입시반 커리큘럼 상세 확인
  ↓
[평가] /instructors → 강사 경력/학력 확인
  ↓
[의사결정] "입시 전문 강사가 있고 합격 실적이 있다"
  ↓
[전환] /contact → 상담 신청 (관심 과정: 입시반)
```

**핵심 터치포인트**: 합격 실적 수치, 입시반 커리큘럼, 강사 프로필

#### 페르소나 B: 취미 수강생

```
[인지] 구글 "기타 레슨 [지역명]" 검색
  ↓
[탐색] / (랜딩) → 학원 분위기 파악 → 수강 후기 확인
  ↓
[비교] /courses → 취미반 커리큘럼 + 시간표 확인
  ↓
[의사결정] "초보자도 가능하고 시간이 맞는다"
  ↓
[전환] /contact → 상담 신청 (관심 과정: 취미반)
```

**핵심 터치포인트**: 수강 후기, 취미반 설명, 시간표, 위치 접근성

#### 페르소나 C: 학부모

```
[인지] 맘카페에서 학원 추천글 발견 → 링크 클릭
  ↓
[탐색] / (랜딩) → 학원 하이라이트 수치 확인
  ↓
[신뢰 구축] /about → 교육 철학, 시설 사진 확인
  ↓
[비교] /instructors → 강사 자격/경력 확인
  ↓
[비교] /courses → 입시반/취미반 과정 차이 확인
  ↓
[의사결정] "체계적이고 시설도 좋다, 자녀를 보내도 되겠다"
  ↓
[전환] /contact → 상담 신청 (문의 내용에 자녀 정보 기재)
```

**핵심 터치포인트**: 시설 사진, 교육 철학, 강사 자격, 수강생 후기

## 4. 단계별 핵심 콘텐츠 & CTA

### 인지 단계 (Awareness)

| 콘텐츠 | 역할 | 위치 |
|--------|------|------|
| Hero 슬로건 + 비주얼 | 첫인상, 브랜드 포지셔닝 | `/` Hero |
| 수치 하이라이트 | 신뢰도 즉시 전달 | `/` HighlightStats |
| **CTA**: "수강 상담 신청" | 즉시 전환 유도 (이미 의사결정한 방문자) | `/` Hero |

### 탐색 단계 (Exploration)

| 콘텐츠 | 역할 | 위치 |
|--------|------|------|
| 커리큘럼 미리보기 카드 | 제공 과목 한눈에 파악 | `/` CurriculumPreview |
| 강사진 미리보기 | 전문성 미리 보기 | `/` InstructorPreview |
| 수강생 후기 | 사회적 증거 (Social Proof) | `/` TestimonialSection |
| **CTA**: 카드 클릭 → 상세 페이지 | 자연스러운 심화 탐색 유도 | 각 카드 |

### 비교/평가 단계 (Evaluation)

| 콘텐츠 | 역할 | 위치 |
|--------|------|------|
| 전공별 커리큘럼 상세 | 교육 내용 구체적 확인 | `/courses` |
| 과정 구분 (입시/취미/전문) | 본인에 맞는 과정 식별 | `/courses` |
| 강사 상세 프로필 | 전문성 검증 | `/instructors` |
| 시설 갤러리 | 환경 신뢰도 | `/about` |
| **CTA**: "맞춤 상담 받기" | 구체적 문의 유도 | `/courses` 수강료 섹션 |

### 의사결정 단계 (Decision)

| 콘텐츠 | 역할 | 위치 |
|--------|------|------|
| FAQ | 마지막 의문점 해소 | `/contact` |
| 수강 절차 안내 | 다음 단계 명확화 | `/courses` |
| 운영 시간/위치 | 실질적 이용 가능성 확인 | `/contact` |
| **CTA**: "무료 상담 예약하기" | 최종 전환 유도 | 각 페이지 CTABanner |

### 전환 단계 (Conversion)

| 콘텐츠 | 역할 | 위치 |
|--------|------|------|
| 상담 신청 폼 | 최종 전환 수단 | `/contact` |
| 전화/카카오톡 바로가기 | 즉시 연락 대안 | `/contact`, 모바일 하단 바 |
| **CTA**: "상담 신청하기" (Submit) | 폼 제출 | `/contact` 폼 하단 |

## 5. 이탈 방지 전략

### 5-1. 페이지별 이탈 방지

| 이탈 지점 | 원인 | 대응 전략 |
|-----------|------|----------|
| Hero 섹션에서 즉시 이탈 | 첫인상 미흡, 관련성 부족 | 강렬한 비주얼 + 명확한 가치 제안 + 즉시 CTA |
| 스크롤 중간 이탈 | 콘텐츠 피로, 관심 저하 | 섹션마다 시각적 변화, 중간 CTA 배치 |
| 수강료 미표시로 이탈 | 가격 정보 부재 불만 | "개인 맞춤 상담" 프레이밍 + 즉시 상담 CTA |
| 상담 폼 이탈 | 폼이 길거나 복잡 | 필수 항목 최소화 (이름, 연락처, 관심 과정만 필수) |
| 모바일 탐색 이탈 | 네비게이션 불편 | 하단 고정 CTA 바, 플로팅 카카오톡 버튼 |

### 5-2. 전역 이탈 방지 요소

1. **Header 고정 CTA**: 스크롤해도 항상 "상담 신청" 버튼 노출
2. **모바일 Sticky Bottom Bar**: 전화 + 카카오톡 2버튼 상시 노출
3. **카카오톡 플로팅 버튼**: 우측 하단 고정, 즉시 채팅 상담
4. **섹션 간 자연스러운 연결**: 각 섹션 하단에 다음 섹션 유도 또는 CTA
5. **소셜 프루프 분산 배치**: 수치, 후기, 성과를 여러 페이지에 분산하여 신뢰 누적

### 5-3. 재방문 유도

- **상담 폼 제출 후**: 감사 메시지 + "카카오톡 채널 추가" 유도
- **SNS 링크**: Footer + 학원 소개 페이지에 인스타그램/유튜브 링크
- **콘텐츠 업데이트**: 수강생 후기, 공연 소식 등 지속적 콘텐츠 추가 (향후 블로그 확장 가능)
