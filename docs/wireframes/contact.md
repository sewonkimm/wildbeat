# 상담 문의 와이어프레임 — `/contact`

## 반응형 Breakpoint 기준

- Mobile: < 768px (1열)
- Tablet: 768px ~ 1023px (2열)
- Desktop: >= 1024px (max-w-7xl 컨테이너)

- **Desktop**: 2열 (폼 60% + 연락처 40%)
- **Mobile**: 1열 스택 (연락처 먼저 → 폼 아래)
- **배경**: background (#0A0A0F)
- **폼 라이브러리**: React Hook Form + Zod validation
- **필수 항목**: 이름, 연락처, 관심 과정, 개인정보 동의
- **선택 항목**: 희망 시간대, 문의 내용
- **제출 후**: 감사 메시지 모달 + "카카오톡 채널 추가" 유도

### 폼 유효성 검증 (Zod Schema)

```
이름: 필수, 2~20자
연락처: 필수, 한국 전화번호 형식 (010-XXXX-XXXX)
관심 과정: 필수, enum 선택
희망 시간대: 선택
문의 내용: 선택, 최대 500자
개인정보 동의: 필수, boolean true
```

- **컴포넌트**: Accordion + AccordionItem
- **동작**: 클릭 시 하나만 열리거나, 복수 열림 허용 (UX 결정 필요)
- **최대 너비**: max-w-3xl, 중앙 정렬
- **배경**: background (#0A0A0F)
- **애니메이션**: Framer Motion height 트랜지션
- **FAQ 항목 수**: 6~8개 (추가 가능)

### 예상 FAQ 항목

| 질문 | 답변 요약 |
|------|----------|
| 수강료는 얼마인가요? | 과정/횟수별 상이, 상담 시 안내 |
| 악기가 없어도 수강할 수 있나요? | 학원 내 악기 사용 가능 |
| 입시 준비는 언제부터 시작하면 좋나요? | 최소 6개월~1년 전 권장 |
| 무료 체험 수업이 있나요? | 상담 시 무료 레벨 테스트 겸 체험 가능 |
| 주차가 가능한가요? | 건물 내 주차 2시간 무료 |
| 수업 시간을 변경할 수 있나요? | 24시간 전 연락 시 변경 가능 |
| 환불 규정은 어떻게 되나요? | 학원 규정에 따름, 상담 시 안내 |
