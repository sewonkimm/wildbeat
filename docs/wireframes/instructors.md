# 강사 소개 와이어프레임 — `/instructors`

## 반응형 Breakpoint 기준

- Mobile: < 768px (1열 또는 2열)
- Tablet: 768px ~ 1023px (2~3열)
- Desktop: >= 1024px (3~4열, max-w-7xl 컨테이너)

- **CTA**: /contact로 이동하며 강사명 파라미터 전달
