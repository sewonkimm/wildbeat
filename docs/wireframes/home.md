# 랜딩페이지 와이어프레임 — `/`

## 반응형 Breakpoint 기준

- Mobile: < 768px (1열)
- Tablet: 768px ~ 1023px (2열)
- Desktop: >= 1024px (3열, max-w-7xl 컨테이너)
