## Section 5: InstructorModal

### 모달 구조

**모달이 열리면 표시되는 정보:**

1. **강사명 및 전공** (H2)
2. **프로필 사진** (1:1, 큰 사이즈)
3. **학력** (H3, bullet list)
4. **주요 경력** (H3, bullet list, 각 항목에 연도 명시)
5. **강사 한마디** (인용문, 블록 스타일)
6. **CTA 버튼**: "이 강사와 상담하기" (강사명 파라미터 전달)

### 모달 동작
- 클릭 시 중앙 오버레이 모달 오픈
- X 버튼, 오버레이 클릭, ESC 키로 닫기 가능
- 모바일에서는 바텀시트 또는 풀스크린 형식으로 변경

## 추가 고려사항

- 강사명, 학력, 경력은 모두 실제 강사 정보로 변경 필수
- 프로필 사진은 통일된 스타일 가이드에 따라 촬영 (조명, 배경 일관성)
- 학력 · 경력의 "OO대" "OO 프로그램" 등은 실명으로 기입
- 강사 한마디는 각 강사의 실제 철학이나 수업 방향을 반영하도록 인터뷰 필요
- 모바일에서 강사 카드는 간소화된 정보만 표시 (프로필 모달에서 전체 정보 확인)
